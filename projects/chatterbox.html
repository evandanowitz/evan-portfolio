Chatterbox Details

Objective

ChatterBox is a native, mobile chat app built using React Native, Expo, and Google Firestore Database, and is optimized for both Android and iOS devices. The app provides users with a chat interface as well as options to share images and a user's location. It also utilizes offline storage with AsyncStorage to allow users to view messages and data while offline.

Technologies / Tools Used

  • React Native
  • Expo CLI
  • Node.js
  • Google Firestore Database
  • Google Firebase Authentication
  • Firebase Cloud Storage
  • Gifted Chat Library
  • Android Studio
